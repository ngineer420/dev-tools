    About this regex tester
    
    
    Type a pattern, toggle the flags you need, and hit Test to see every match highlighted in your test string along with each match's index and capture groups. Invalid patterns are reported instead of failing silently, and named groups are shown when you use them. Everything runs locally in your browser using the native JavaScript regex engine — nothing you paste is uploaded. For anchors, quantifiers, and how to avoid catastrophic backtracking, read the regex cheat sheet.
